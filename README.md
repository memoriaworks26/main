## 목업 실행

```bash
cd 개발/앱
npm install      # 최초 1회
npm run dev      # → http://localhost:5173/
```

> Tailwind는 Play CDN(`index.html`), 폰트는 인라인 `@import`(Pretendard·Nanum Myeongjo). 본개발에선 tailwind 빌드 + TS로 교체.

### 구현 범위 (전체 인터페이스 · 더미데이터)
- **관리자**: 현황판 · 빈소·호실 상세 · 사이니지 · 고객관리 · 예약·접수 · 편집·컨펌 · 콘텐츠 허브 · 외부 링크 · 정산(파트너사별/건별·**거래명세서**·파트너사 상세) · 환경설정(템플릿·AI프롬프트·BGM·알림·파트너사권한·디바이스·회사정보) · 시스템(잡 큐·업타임) · 스토리지
- **파트너**(자사 스코핑): 통합 대시보드 · 예약 접수(확정→보호자 URL 생성) · 예약 목록·**상세** · 라이브 컨트롤 · 편집·컨펌 · 콘텐츠 허브 · 외부 링크 · 정산 조회 · 설정
- **유저 링크**: 모바일 7단계 위저드
- **영상 편집기**: 좌 블록리스트(순서 고정) / 중앙 미리보기(서버 렌더 재생)+트림+타임라인 / 우 블록타입별 편집 패널 — 각 화면의 "컨펌/편집기 열기"로 진입

> 설계는 `기획서.md`(아이보리/네이비/골드·명조·빨강 배제·lucide / IA)를 따랐습니다. (클라이언트 시뮬레이터의 기능·플로우 디테일은 이미 반영 완료 — 펫 도메인·빨강·이모지는 미반영.)

## 본개발 이관

1. **먼저 읽기**: `개발/기획서.md` (전부 여기에).
2. Vite + React(**TS**) + Tailwind(빌드)로 올리고 `앱/src`의 컴포넌트를 그대로 이식.
3. 로고: 알파 살아있는 원본 PNG/SVG를 `src/assets`에 두고 import (현재 b64는 임시).
4. 더미데이터(`data.js`) → 백엔드 API 연결. 디자인·구조는 그대로 살림.
