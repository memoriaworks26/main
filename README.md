## IDE/Claude Code에서 이어가는 법

1. **먼저 읽기**: `01_기획/개발착수전_정리.md` → `05_대화기록/프로젝트_맥락_요약.md` → `03_목업_프론트/디자인_가이드.md` 순.
2. **프론트 시작**: Vite + React(TS) + Tailwind 프로젝트 생성 → `현황판_목업.jsx`를 컴포넌트로 분리 이식 (디자인 가이드의 "프론트 이관" 절 참고).
3. **로고**: 가능하면 진짜 알파 채널 살아있는 원본 PNG/SVG를 받아 `src/assets`에 넣고 import (지금 b64는 임시).
4. **폰트**: `pretendard` npm 패키지 + Nanum Myeongjo(고인 성함용) 설치.
5. **다음 화면**: 영상 편집기 + 컨펌 화면 (좌 블록리스트 / 중앙 미리보기+트림 / 우 블록별 편집패널).
6. **백엔드**: 명세서·정리.md의 스택대로. 목업의 더미 데이터(ROOMS)를 API 연결로 교체.

> 주의: 이 목업은 "화면 구조·톤 확정"이 목적이라 단일 파일·CDN 폰트·더미 데이터입니다. 본개발에선 구조·디자인은 그대로 살리고 데이터 연결·컴포넌트 분리만 추가하면 됩니다(화면 다시 안 만듦).
